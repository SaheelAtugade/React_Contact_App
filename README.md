# React Contact Manager

A simple React project built while learning React fundamentals. This project allows users to add and delete contacts using a clean UI.

## 🚀 Concepts Learned

- React Components
- Props
- Props Drilling
- useState Hook
- Conditional Rendering
- Two-Way Binding
- Form Handling
- State Lifting
- Event Handling
- Rendering Lists with `map()`
- Deleting Items with `filter()`
- Passing Functions as Props
- Dynamic Rendering
- Component Reusability
- Responsive UI

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- Tailwind CSS

## 📌 Features

- Add new contacts
- Delete contacts
- Default profile image support
- Empty state message
- Responsive and clean UI
